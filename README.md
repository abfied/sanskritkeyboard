# Sanskrit Keyboard - संस्कृत कीबोर्ड

A modern, responsive web application for typing in Sanskrit using Devanagari script.

## Features

- Complete Devanagari character set
- Vowels, consonants, vowel signs, and special characters
- Devanagari numbers
- Copy to clipboard
- Download as text file
- Responsive design for all devices
- Beautiful UI with Indian temple illustrations

## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
